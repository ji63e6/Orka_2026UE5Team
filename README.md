# Orka_2026UE5Team
오르카 2기 2026-1 언리얼엔진 팀 github 레포지토리 입니다!

**Github란?**
다같이 클라우드를 공유하는 느낌이라고 생각해주시면 될 것 같습니다. 다같이 에셋이나 파일을 공유할 때 카카오톡으로 공유하는데에는 한계가 있을 듯 해서 깃을 활용하면 좋을 것 같습니다.
다만 다같이 공유하는거다 보니까 누군가 잘못 파일을 지우거나 덮어쓸 경우 **모두의 파일이 지워집니다** 되돌리는 방법은 저도 모릅니다.. 저도 깃을 잘 안써봐서.. 사용방법에 매우 무지해서 그때그때 인터넷을 참고하면서 써야합니다. 
그렇기 때문에 뭔가 파일이 잘못될 경우 복구방법이 없으므로 다들 주의해서 사용 부탁드립니다.

**모두가 할 일**
1. github 계정을 만들고 단톡방에 github 아이디를 보내주시면 초대해드리겠습니다!
2. gitkraken을 다운받고 설치합니다.
3. gitkraken에 로그인합니다.
4. github repository에 초대되었다면 우측 상단의 Fork 를 누릅니다. (각자 계정으로 복제하는 개념이라고 생각해주시면 됩니다.)
5. create fork를 해줍니다 >> 모든 작업은 각자의 fork된 레포지토리에 업로드해주시면 됩니다.
6. fork된 repository > code 누르고 링크를 복사해줍니다
7. gitkraken에서 file > clone a repo 를 누르고 url에 복사된 링크를 붙여넣습니다.
8. 편한 폴더를 새로 생성하고 where to clone to 에서 해당 주소를 선택 후 clone 버튼을 눌러줍니다.
   >> 이렇게 하시면 그 해당 폴더가 깃헙과 연동된다는 느낌으로 보시면 됩니다. 해당 폴더에 어떤 파일을 넣으면 깃헙에 자동으로 올라가고, 폴더에서 파일을 지우면 파일이 지워지는 겁니다.
